# Community Tools Quick Reference

## üéØ Top 5 Most Community-Useful Tools

### 1. PR Summarizer
**Best for:** Open source communities, code review teams
**Setup time:** 5 minutes
**Value:** Saves hours of PR review time
**Quick start:**
```bash
cd pr-summarizer
pip install -r requirements.txt
python pr_bot.py --repo owner/repo --pr 123
```

### 2. Meeting Action Extractor
**Best for:** Community meetings, project teams
**Setup time:** 2 minutes
**Value:** Never lose track of action items
**Quick start:**
```bash
cd meeting-action-extractor
python extract_actions.py -i meeting_notes.txt -o actions.json
```

### 3. Dead Link Checker
**Best for:** Documentation sites, community wikis
**Setup time:** 1 minute
**Value:** Maintains website quality
**Quick start:**
```bash
cd dead-link-checker
python checker.py https://your-community-site.com
```

### 4. GitHub Star Notifier
**Best for:** Open source maintainers
**Setup time:** 3 minutes
**Value:** Engage with new community members
**Quick start:**
```bash
cd github-star-notifier
export GITHUB_TOKEN=your_token
export GITHUB_REPO=owner/repo
python notify.py --check-once
```

### 5. Post-Mortem Generator
**Best for:** DevOps teams, incident response
**Setup time:** 1 minute
**Value:** Standardized incident documentation
**Quick start:**
```bash
cd postmortem-generator
python generate_postmortem.py --interactive
```
